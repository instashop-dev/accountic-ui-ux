# Accountic — Design System

> **Retired.** The "Ethereal" glassmorphism system documented here has been
> replaced by the **Cobalt** design system.
>
> **The authoritative spec now lives at [`/design.md`](../../design.md)**, with
> tokens in [`/src/styles/tokens.css`](../../src/styles/tokens.css).

Cobalt in one paragraph: cool engineered near-white paper, one electric-cobalt
signal accent used on < 5% of any viewport, hairline structure (no glassmorphism,
no drop shadows), Space Grotesk display + Inter body + JetBrains Mono labels/data,
tight 6–10px radii, and one dark graphite band per page. Nav is a flush bordered
bar with a working ⌘K command palette; the footer is a closing statement + hairline
link columns.

See [`/design.md`](../../design.md) for the full token set, typography, component,
motion, and per-page specifications. Do not add rules here — amend `design.md`.
